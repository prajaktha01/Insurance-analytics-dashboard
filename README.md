Project Overview

This repository contains a complete Insurance Analytics Dashboard developed using Excel, Power BI, Tableau, and SQL.
The goal of this project is to analyze insurance business performance across Cross Sell, New Business and Renewal using interactive visualizations and KPIs.

The same dashboard is built in three tools to demonstrate multi-tool analytics skills.



Tools Used
	•	Power BI – Interactive visuals, DAX measures
	•	Tableau – Advanced charts & story points
	•	Excel – Pivot Tables, Pivot Charts, Slicers
	•	SQL – Data cleaning, transformations, and KPIs calculation


  Dashboard Objectives
	•	Compare Target vs Achieved performance
	•	Analyze revenue by business segment
	•	Track meeting performance by Account Executives
	•	Identify top revenue opportunities
	•	Understand product distribution across insurance types
	•	Provide filters for dynamic analysis (Year, Stage, Income Class, AE)



 Key Features

	•	Slicers for Year, Executive, Stage
	•	KPIs for Cross Sell, New, Renewal

Tableau

SQL

 How to Use This Project

Excel
	1.	Download .xlsx file
	2.	Click Enable Content
	3.	Use filters to explore the dashboard

Power BI
	1.	Download .pbix file
	2.	Open in Power BI Desktop
	3.	Explore pages & slicers

Tableau
	1.	Download .twbx file
	2.	Open in Tableau Desktop

SQL
	1.	Open .sql file
	2.	Run queries to reproduce transformations
